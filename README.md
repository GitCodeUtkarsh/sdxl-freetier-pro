# SDXL FreeTier Pro 🚀

An optimized Google Colab notebook for running SDXL text-to-image generation with optional LoRA support using ComfyUI backend workflows.

Designed specifically for Google Colab Free Tier.

## ✨ Features

- 🎨 SDXL Text-to-Image Generation
- 🔥 LoRA Model Support
- ⚡ Free-Tier Colab Optimized
- 🧠 ComfyUI Backend Execution
- 🚫 No ComfyUI Web UI Required
- 🖼️ Direct Image Preview in Notebook
- 📦 Automated Setup
- 🎯 Beginner Friendly

## 💡 Why This Project?

Running the full ComfyUI interface on Google Colab Free Tier can be inefficient or restricted.

This notebook uses ComfyUI backend workflows directly, reducing overhead while maintaining SDXL image generation capabilities.

## 🚀 Getting Started

1. Open the notebook in Google Colab
2. Run all cells
3. Select your SDXL model
4. Optionally load a LoRA
5. Enter your prompt
6. Generate images
